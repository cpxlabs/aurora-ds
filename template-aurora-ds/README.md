# Template Aurora DS

App base para novos produtos com Aurora DS.

## O que já vem pronto

- Layout compartilhado com menu lateral e header
- Fluxos de login, cadastro e perfil
- Estrutura de estilos global para reaproveitamento

## Rodar localmente

```bash
pnpm install
pnpm --dir template-aurora-ds dev
```

## Build

```bash
pnpm --dir template-aurora-ds build
```

## Deploy na Vercel

Use o `template-aurora-ds/vercel.json` e configure a raiz do repositório no projeto da Vercel.
