`Textarea` é um campo de texto multilinha construído sobre o elemento nativo `<textarea>`.

- Segue o mesmo padrão visual e de API do componente `Input`
- Prop `invalid` adiciona borda vermelha e `aria-invalid="true"` para acessibilidade
- Redimensionamento vertical habilitado por padrão
- Aceita `disabled`, `rows` e todas as outras props nativas de textarea
</Description>
