<Description>
`Spinner` é um indicador de progresso indeterminado para estados de carregamento.

- Tamanhos: `sm`, `md` (padrão) e `lg`
- Usa `role="status"` e `aria-label` para acessibilidade com leitores de tela
- Animação puramente via CSS, sem dependências adicionais
</Description>
