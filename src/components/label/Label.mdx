<Subtitle>Rótulo acessível para campos de formulário</Subtitle>

<Description>
`Label` é um wrapper do elemento nativo `<label>` com estilo alinhado aos tokens do Aurora DS.

- Use a prop `htmlFor` para associar o rótulo ao campo correspondente
- A prop `invalid` aplica a cor de erro (`--ui-negative`)
</Description>

}`}
