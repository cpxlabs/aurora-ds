<Description>
`Alert` é um componente de notificação inline para exibir mensagens de feedback ao usuário.

- Tons: `neutral`, `positive`, `negative` e `warning`
- Usa `role="alert"` para acessibilidade
- Componha com `AlertTitle` e `AlertDescription` para estrutura semântica
</Description>
