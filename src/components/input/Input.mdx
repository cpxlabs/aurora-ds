<Description>
`Input` é um campo de entrada construído sobre o elemento nativo `<input>`.

- Suporta todos os tipos HTML (`text`, `email`, `password`, `number`, etc.)
- Prop `invalid` adiciona borda vermelha e `aria-invalid="true"` para acessibilidade
- Aceita `disabled` e todas as outras props nativas de input
</Description>
