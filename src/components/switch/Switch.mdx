`Switch` é construído sobre `@radix-ui/react-switch` com estilo alinhado ao Aurora DS.

- Totalmente acessível via teclado (barra de espaço para alternar)
- Suporta estados controlado e não controlado
- Use junto com `Label` para associar um rótulo clicável
