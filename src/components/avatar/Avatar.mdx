<Subtitle>Representação visual de um usuário com imagem ou iniciais</Subtitle>

<Description>
`Avatar` exibe a foto de perfil de um usuário. Quando nenhuma imagem é fornecida, mostra as iniciais como fallback.

- Tamanhos: `sm`, `md` (padrão) e `lg`
- Prop `src` define a URL da imagem; prop `initials` define o fallback de texto
- Acessível via `role="img"` e `aria-label`
</Description>
