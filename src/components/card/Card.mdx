<Description>
`Card` é um contêiner flexível construído com CSS Modules e tokens do Aurora DS.

- Use `CardMedia` para cabeçalhos com imagem
- Monte layouts com `CardHeader`, `CardContent`, `CardFooter` e `CardAction`
- Aplique tons de badge com `neutral`, `positive` e `negative`
- Renderize de forma polimórfica com `asChild`
</Description>
