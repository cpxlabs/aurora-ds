<Title>Separator</Title>
<Subtitle>Divisor visual horizontal ou vertical entre seções de conteúdo</Subtitle>

<Description>
`Separator` é um divisor visual construído sobre o elemento `<hr>`.

- Orientações: `horizontal` (padrão) e `vertical`
- Usa `role="separator"` e `aria-orientation` para acessibilidade
- Renderiza com 1px de espessura usando o token `--ui-border`
</Description>
