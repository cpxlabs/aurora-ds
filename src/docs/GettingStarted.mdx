Este Storybook documenta os componentes da biblioteca **my-ui**.

### Instalação

```bash
npm i @sua-scope/my-ui
```

### Importação

> A biblioteca exporta tudo no **root**.

```ts
import { Button, Dialog } from '@sua-scope/my-ui';
```

### Dark mode

Use o atributo **`data-theme="dark"`** no elemento `<html>` (ou em um wrapper):

```html
<html data-theme="dark"></html>
```

No Storybook, você pode alternar o tema no botão **Tema** (Light/Dark) na toolbar.
</Description>
