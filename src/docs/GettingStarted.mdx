Este Storybook documenta os componentes públicos do Aurora DS.

### Instalação

```bash
pnpm add @cpxlabs/aurora-ds
```

### Importação

> Importe os componentes pelo pacote raiz. O entrypoint público já carrega os tokens de tema automaticamente.

```ts
import { Button, Dialog } from '@cpxlabs/aurora-ds';
```

Se você quiser expor apenas os estilos de tema, a entrada `@cpxlabs/aurora-ds/theme.css` continua disponível.

### i18n

Use `AuroraI18nProvider` para traduzir os textos internos padrão dos componentes para `pt_br`, `es` ou `eng`.

```tsx
import { AuroraI18nProvider, Dialog, DialogContent, DialogTitle } from "@cpxlabs/aurora-ds";

export function ExemploComIdioma() {
  return (
    <AuroraI18nProvider locale="es">
      <Dialog open>
        <DialogContent>
          <DialogTitle>Confirmación</DialogTitle>
        </DialogContent>
      </Dialog>
    </AuroraI18nProvider>
  );
}
```

### Modo escuro

Use o atributo `data-theme="dark"` no `<html>` ou em um wrapper:

```html
<html data-theme="dark"></html>
```

No Storybook, você pode trocar o tema pelo controle **Tema** na toolbar.
