<Title>Como usar componentes</Title>
<Subtitle>Padrões práticos para montar interfaces com o Aurora DS</Subtitle>

<Description>
Use os componentes do Aurora DS pelo pacote raiz e monte a interface por composição.

- o pacote raiz já carrega os tokens de tema automaticamente;
- reutilize componentes compostos como `Card`, `Alert`, `Tabs` e `Dialog`;
- prefira manter `Label`, `Input`, `Select`, `Checkbox` e `Switch` conectados por `id` e `htmlFor`.
</Description>

## Fluxo básico

<Source
  language="tsx"
  code={`import {
  Button,
  Card,
  CardContent,
  CardDescription,
  CardHeader,
  CardTitle,
  Input,
  Label,
} from "@cpxlabs/aurora-ds";

export function LoginCard() {
  return (
    <Card>
      <CardHeader>
        <CardTitle>Acessar conta</CardTitle>
        <CardDescription>Monte a tela combinando componentes públicos.</CardDescription>
      </CardHeader>

      <CardContent style={{ display: "grid", gap: 12 }}>
        <div style={{ display: "grid", gap: 6 }}>
          <Label htmlFor="email">E-mail</Label>
          <Input id="email" type="email" placeholder="voce@empresa.com" />
        </div>

        <Button>Entrar</Button>
      </CardContent>
    </Card>
  );
}`}
/>
