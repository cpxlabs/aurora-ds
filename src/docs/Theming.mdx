<Subtitle>Temas claro e escuro com variáveis CSS do Aurora DS</Subtitle>

<Description>
Os componentes consomem os tokens definidos em `src/theme.css`. A troca de tema é feita por atributo:

- **Claro**: valores padrão em `:root`
- **Escuro**: sobrescritas em `[data-theme="dark"]`

Isso mantém o tema independente de framework para as aplicações consumidoras.
</Description>

## Como alternar o tema em um app

<Source
  language="tsx"
  code={`import "@cpxlabs/aurora-ds/theme.css";

export function setTheme(theme: 'light' | 'dark') {
  const root = document.documentElement;
  if (theme === 'dark') root.setAttribute('data-theme', 'dark');
  else root.removeAttribute('data-theme');
}`}
/>
