<Subtitle>Light/Dark via CSS variables e `[data-theme="dark"]`</Subtitle>

<Description>
Os componentes usam tokens definidos em `src/theme.css` (CSS variables). O tema é controlado por atributo:

- **Light**: padrão em `:root`
- **Dark**: sobrescrito em **`[data-theme="dark"]`**

Isso permite trocar o tema sem depender de Tailwind/Styled Components.
</Description>

## Como alternar tema no app

<Source
  language="tsx"
  code={`export function setTheme(theme: 'light' | 'dark') {
  const root = document.documentElement;
  if (theme === 'dark') root.setAttribute('data-theme', 'dark');
  else root.removeAttribute('data-theme');
}`}
/>
